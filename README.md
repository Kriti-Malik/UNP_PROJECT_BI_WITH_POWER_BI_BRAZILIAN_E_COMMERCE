# UNP_PROJECT_BI_WITH_POWER_BI_BRAZILIAN_E_COMMERCE

I did this project in collaboration with UNP, where I designed and implemented an e-commerce data management and analytics system using SQL and Power BI to support efficient operations and enable data-driven decision-making.
Designed a well-structured relational database schema by identifying key entities and defining relationships between them, while implementing primary and foreign key constraints to ensure data integrity, consistency, and efficient querying across interconnected tables such as Products, Customers, Orders, Sales Returns, Payments, Reviews, Employees, and Shipments.
Developed and managed the complete database by performing data insertion and simulating real-world business scenarios including order processing, payment tracking, shipment handling, and product returns, ensuring the system accurately reflected real e-commerce workflows and operations.
Conducted in-depth data analysis using optimized SQL queries to extract insights such as monthly sales trends, top-selling products, high-value customers, and the impact of discounts and returns on revenue, while also identifying patterns in customer purchasing behavior, order frequency, and operational inefficiencies like delayed shipments.
Built interactive Power BI dashboards with charts, KPIs, slicers, and drill-down features, enabling dynamic data exploration, visualization of key metrics, tracking of business performance, identification of trends and patterns, and supporting strategic, data-driven decision-making.
